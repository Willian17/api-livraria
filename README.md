# Livraria
## Livraria é um projeto de gerenciamento de uma livraria feita com Java e Spring Boot

### Especificações técnicas
<ul>
  <li> Java 11 </li>
  <li> Spring Boot 2.5.1 </li>
</ul>

### Recursos
<ul>
  <li> Database pagination </li>
  <li> Autenticação </li>
  <li> Caching </li>
  <li> Swagger (documentação da api) </li>
</ul>
